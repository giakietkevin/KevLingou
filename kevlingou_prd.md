# Kevlingou - Product Requirements Document (PRD)

## Project Overview
**Name:** Kevlingou
**Description:** A minimalist web platform for self-studying Japanese (N5 to N2), utilizing Spaced Repetition (SRS) and focus-oriented UI/UX.

## Goals
- **Learning:** Clear roadmap from alphabets (Hiragana/Katakana) to JLPT levels with Vocabulary, Grammar, Kanji, Listening, and Reading.
- **System:** Smooth web application with visual progress tracking and automated daily review suggestions.

## Core Features (MVP)
1. **Learning System:** Level-based modules (Alphabets, N5, N4...) including Vocab, Kanji, and Grammar.
2. **Flashcard & SRS:** Spaced Repetition System (Anki-style) for optimal memory retention.
3. **Progress Management:** Minimalist Dashboard with Streaks, % completion, and vocab count.
4. **Exercises:** Small tests (MCQs, fill-in-the-blanks, sentence building) after each module.
5. **AI Conversation Assistant (Should-have):** Chatbot for real-time Japanese conversation practice and grammar correction.
6. **Kanji Analysis (Could-have):** Radical breakdown for easier memorization.

## UI/UX Requirements
- **Minimalism:** High white-space, neutral colors, low eye strain.
- **Responsive:** Desktop and Mobile functionality.
- **Gentle Gamification:** GitHub-style Heatmap for tracking daily study habits.

## Technical Stack
- **Frontend:** React + Tailwind CSS.
- **Backend:** C# .NET or Node.js.
- **Database:** SQL Server or PostgreSQL.
- **AI:** Python (FastAPI/Flask) for conversation and OCR.
